# Smart music stand

It comes with a metronome and a system that reads the metronome.

based on the number of bars on a row, and the number of rows, it gets down to the page (pdf, image, etc) on a speed based on the BPM, and goes to the next tab naturally...

Or it could just start going lower based on the BPM

The landing page should have like buttons to start a session, and also ways to start a pre-figured session. 

Other features I'm thinking
In practice: 
1. play button 
2. Human vocals start counting on the bpm
3. The song starts from the configured part the user chose 

Instrument-less version: guitarless for example, and when the song starts, it also records the user's play, and gets saved. Later the user can hear the song with his guitar.
